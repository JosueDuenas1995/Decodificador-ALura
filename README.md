
# <p align="center">Encriptador de Texto</p>
  
Este es un desafio propuesto para el Grupo N7 del proyecto ONE de Alura LATAM, se propuso la creación de un decodificador de texto, basado en un modelo propuesto en Figma,en este caso, para mi proyecto, tomé la decisión de usar el diseño propuesto e incluir un par de funciones extra al programa.
##  Caracteristicas del Programa   
El programa consiste en un sitio web que permita encriptar y desencriptar textos, validar que solo se ingresen letras minúsculas y sin acentos, y realizar acciones de copiar, intercambiar y resetear textos.

Para eso se usó el lenguaje Javascript , Las "llaves" de encriptación que fueron usadas para el programa fueron las siguientes:

- La letra "e" es convertida para "enter"
- La letra "i" es convertida para "imes"
- La letra "a" es convertida para "ai"
- La letra "o" es convertida para "ober"
- La letra "u" es convertida para "ufat"

Al momento de escribir una palabra, por ejemplo, si escribes “Daniel” el programa mostrara una alerta indicando que no puedes colocar Mayúsculas o caracteres especiales  “daniel”, el programa lo convertirá en “dainimesenterl” y con la opción copiar aparecerá una alerta indicando que se copió exitosamente en portapapeles.

## Requisitos del Proyecto 

- Debe funcionar solo con letras minúsculas 
- No deben ser utilizados letras con acentos ni caracteres especiales Debe ser posible convertir una palabra para la versión encriptada también devolver una palabra encriptada para su versión original.
- La página debe tener campos para inserción del texto que será encriptado o desencriptado, y el usuario debe poder escoger entre ambas opciones.
- El resultado debe ser mostrado en la pantalla.

##  Extras
- Un botón que copie el texto encriptado/desencriptado 
- Un botón de reseteo para limpiar la pantalla y escribir datos Nuevamente
- Un botón de Intercambio que intercambiara la posicion de los textos encriptados y desencriptados.
